1. Introduction

SpendWise is a mobile application designed to help users track expenses, set budgets, monitor savings goals, and gain insight into their financial habits.

Our goal is to make personal finance easier for both individuals and groups by providing clear visual dashboards, reminders, and intelligent insights.

  • Log daily expenses quickly.
  • Create and track monthly budgets.
  • Visualize progress using Savings Circles.
  • Chat with an AI-based financial assistant.
  • Toggle between Light and Dark modes for better readability.

2. Design & Architecture

SpendWise follows an MVVM (Model–View–ViewModel) architecture to keep the UI and business logic clearly separated. The app uses:

  • ViewModel + LiveData for reactive UI updates.
  • Repository pattern to abstract data sources.
  • Firebase Firestore / Room for persistent storage.
  • SharedPreferences to cache theme and user preferences locally.
  • RecyclerView for lists like expenses and budgets.

5. Conclusions, Reflections, and Learning

Building SpendWise gave our team hands-on experience with Android development, Git-based collaboration, and clean architecture patterns.

Some of the key challenges we faced included:

  • Managing state across multiple fragments while keeping the UI responsive.
  • Ensuring theme changes (Light/Dark Mode) propagate instantly across screens.
  • Designing a database schema that works well with both budgets and expenses.

Through this project, we learned:

  • How to apply MVVM with ViewModel and LiveData in a real app.
  • How to integrate Firebase/Room and SharedPreferences effectively.
  • How to design consistent UI/UX with reusable components and themes.

In the future, we plan to extend SpendWise with advanced analytics, export to CSV, and optional bank account integration.
